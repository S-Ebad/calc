# calc

A command-line calculator written in Rust. Supports variables, a rich set of math functions, and a natural expression syntax.

## Build & Run

```
cargo run
```

Type expressions at the `>` prompt. Press `Ctrl+C` or `Ctrl+D` to exit.

## Usage

```
> 2 + 3
= 5

> x = 10
= 10

> x^2 + 2x + 1
= 121

> sin(pi / 2)
= 1

> 5!
= 120
```
